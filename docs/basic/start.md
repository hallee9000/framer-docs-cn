# 开始
从基础知识开始学习Framer。
### 1. 图层
图层可以是图像、文本或其他。它们拥有层级关系，通过各种属性来定义它们的位置、外观和交互。
[图层详细说明](/docs#layer)
### 2. 动画
让图层的大小、颜色等属性在变化时形成过渡动画，可以使用不同的弹性曲线或缓动效果，也可以设置动画的时间或延迟等属性。
[动画详细说明](/docs#animation)
### 3. 状态
一个状态可以定义一系列不同的属性值。你可以添加、移除或者变换状态，带有动画效果或者不带。
[状态详细说明](/docs#states)
### 4. 事件
事件用来监测和响应用户的操作、动画的开始或结束、变量值的改变等。
[事件详细说明](/docs#events)
